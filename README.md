# AI Systems, Architecture & Applied Intelligence

### by Anirban Dutta

Designing, building, and scaling intelligent systems — from foundational principles to real-world AI architectures.

---

## Overview

This repository represents a structured body of work authored by Anirban Dutta, covering the full spectrum of artificial intelligence — from core computational foundations to advanced, production-grade intelligent systems.

The content is designed to move beyond theory into **practical system design, architecture, and real-world application**.

---

## What This Collection Covers

* **Foundations** → Data Structures, Algorithms, Computational Thinking
* **Machine Learning** → Core principles, training, optimization
* **Deep Learning** → Neural architectures and representation learning
* **Generative AI** → Modern LLMs, prompting, and system integration
* **AI Systems** → MLOps, DevOps for AI, scalable pipelines
* **Agentic AI** → Autonomous and multi-agent system design
* **Advanced Topics** → Governance, alignment, and post-AGI systems

---

## Recommended Learning Path

For a structured progression through the material:

1. Data Structures & Algorithms
2. Machine Learning Fundamentals
3. Deep Learning & Neural Architectures
4. Generative AI
5. AI Systems & MLOps
6. Agentic & Multi-Agent Systems
7. Advanced AI (Governance, Alignment, Post-AGI)
